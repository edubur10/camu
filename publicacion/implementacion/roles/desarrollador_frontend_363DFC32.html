Role: desarrollador frontend
Un desarrollador frontend en un proyecto de software es responsable de crear y mantener la interfaz de usuario y la experiencia del usuario en la parte visible de una aplicación o sitio web. Su objetivo principal es traducir los diseños y requisitos funcionales en código HTML, CSS y JavaScript que los usuarios puedan interactuar directamente.
Relationships
Primary Performs
  • conectar controlador con la interfaz
  • crear controlador de la interfaz
  • crear entidades
  • crear interfaz
  • crear metodos asociados a la interfaz
  • crear metodos de persistencia
  • crear metodos en el controlador de la interfaz
  • crear variables asociadas a widget
  • identificar funcionalidades asociadas a la interfaz
  • identificar puntos claves de la interfaz
Modifies
  • interfaz de usuario
  • interfaz y controlador asociados
  • programa funcional
