Artifact: listado de requisitos detallados
Un listado de requisitos detallados es un documento que enumera y describe de manera precisa y completa todos los requisitos funcionales y no funcionales de un sistema de software. Estos requisitos se especifican en detalle para proporcionar una comprensión clara de lo que se espera que el sistema haga y cómo debe comportarse.
