Role: administrador base de datos
El administrador de la base de datos (DBA, por sus siglas en inglés: Database Administrator) es el profesional encargado de administrar y gestionar una base de datos dentro de un sistema de software. El rol del administrador de la base de datos es crítico para asegurar la integridad, disponibilidad, seguridad y rendimiento de la base de datos.
Synonyms: Gestor de bases de datos
