Role: desarrollador frontend
Un desarrollador frontend en un proyecto de software es un profesional encargado de crear la interfaz de usuario y la experiencia de usuario en la parte del cliente de una aplicación o sitio web. Su principal objetivo es traducir el diseño visual y funcionalidades establecidos
Relationships
Primary Performs
  • diseñar aspecto visual
  • establecer validaciones y restricciones
  • prototipar la interfaz
  • realizar pruebas de usabilidad
Additionally Performs
  • crear bocetos y esquemas iniciales
  • definir la disposicion de los campos
Modifies
  • interfaces de usuario
