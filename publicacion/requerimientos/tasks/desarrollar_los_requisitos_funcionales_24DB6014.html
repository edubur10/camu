Task: desarrollar los requisitos funcionales
Desarrollar los requisitos funcionales de un proyecto de software implica identificar, analizar y documentar las funciones y características específicas que el software debe cumplir para satisfacer las necesidades y expectativas de los usuarios y las partes interesadas. Los requisitos funcionales describen qué acciones debe realizar el sistema, cómo debe responder a las entradas del usuario y qué resultados se esperan. Estos requisitos suelen expresarse en términos de comportamiento del sistema y funciones específicas que deben estar presentes.
