Artifact: requisitos funcionales
Los requisitos funcionales de software son declaraciones detalladas que describen las funciones, las acciones y los comportamientos específicos que un sistema de software debe cumplir para satisfacer las necesidades de los usuarios y las partes interesadas. Estos requisitos se centran en qué debe hacer el software y cómo debe responder a las entradas del usuario.
